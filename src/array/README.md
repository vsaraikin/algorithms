Subarray - [3, 6, 1] is subarray of [2, **3, 6, 1**, 5, 4] (contiguous elements). 

Subsequence - [3, 1, 5] is a subsequence of [2, **3**, 6, **1**, **5**, 4] (noncontiguous elements).
